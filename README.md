YT Speed

Remember your YouTube speed, set custom playback rates above 2x, and add
video delay to match Bluetooth audio latency.

## Install

1. Open `chrome://extensions`.
2. Turn on Developer mode.
3. Click Load unpacked.
4. Select this folder.

## Use

1. Open a YouTube video page.
2. Click the extension icon.
3. Set playback speed (or use presets) and click Apply.
4. Set Bluetooth video delay in milliseconds and click Apply.
5. Optional: choose the mic source from "Mic input", then click "Auto
   Calibrate With Mic" to estimate delay from speaker output. Keep the popup
   open, allow mic access, and keep the selected mic near the speaker until
   calibration finishes.
6. Optional fallback: click "Start Popup Calibration" and use the manual sweep
   preview if auto calibration cannot detect a stable result.
